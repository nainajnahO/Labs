# MIPS Pipeline Lab

Coursework for **Computer Architecture (DARK)** — course 1DT038, Uppsala University, Spring 2022.

## Contents

| File | Description |
|------|-------------|
| `string_functions.s` | MIPS assembly: integer array sum and string-handling routines |
| `testcode.s` | Small MIPS test program (NOPs inserted to avoid pipeline hazards) |
| `mips-pipeline-lab.circ` | Logisim model of a pipelined MIPS processor |
| `*.mem` | Memory image files used to test the circuit |

## Tools

- **MARS** (or SPIM) — to assemble and run the `.s` files
- **Logisim** — to open `mips-pipeline-lab.circ` and load the `.mem` files
